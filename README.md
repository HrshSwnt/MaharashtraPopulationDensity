# Summary
Easy way to access Sub District level population density map of Maharashtra.
Viewable at https://hrshswnt.github.io/MaharashtraPopulationDensity/

# Instructions
The map can be generated by running the python notebook.
The excel data was converted to csv with pandas and cleaned up or easier parsing.
